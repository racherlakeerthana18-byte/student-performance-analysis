# Student Academic Performance Analysis

This project analyzes student academic performance data using Python and NumPy.
It calculates student-wise and subject-wise averages and identifies top and low performers.

## Features
- Calculates average marks for each student
- Calculates average marks for each subject
- Identifies highest and lowest performing students
- Uses NumPy for numerical analysis

## Technologies Used
- Python
- NumPy

## How to Run the Project
1. Install Python (version 3.x)
2. Install NumPy:
   pip install numpy
3. Run the script:
   python analysis.py

## Sample Output
- Average marks per student
- Subject-wise averages
- Highest and lowest performing students
